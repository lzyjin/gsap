ScrollTrigger 사용

scrollTrigger: 스크롤을 해서 해당 요소가 보일 때 애니메이션이 나타나도록 하기

scrollTrigger속성값: 애니메이션 실행 기준이 되는 요소

		gsap.to('.c', {
			scrollTrigger: '.c',
			x: 400,
			rotation: 360,
			duration: 3
		});
		---> .c 요소가 보이기 시작할 때 .c의 애니메이션이 실행된다. 
	
		gsap.to('.b', {
			scrollTrigger: '.c',
			x: 400,
			rotation: 360,
			duration: 3
		});
		---> .c 요소가 보이기 시작할 때 .b의 애니메이션이 실행된다.

toggleActions: 4개의 토글위치에서 순서대로 제어되는 방식을 결정한다

onEnter, onLeave, onEnterBack, onLeaveBack 순서

들어갈 때, 나갈 때, 뒤로 다시 들어갈 때, 처음부터 끝까지 스크롤할 때 순서

default: play none none none
