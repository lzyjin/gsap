gsap 기초

Common methods for creating a Tween (all of these methods return a Tween instance):
gsap.to(), gsap.from(), gsap.fromTo()

Tween은 애니메이션이 작동하도록 하는 객체


gsap.to()

gsap.to(대상(querySelector 문법), 속성들)

return : Tween

    속성
  • x: x축 방향 이동값
  • rotation: 회전값
  • duration: 지속시간 (초, default: 0.5)
  • opacity: 투명도 (default: 1)
  • delay: 딜레이 (초)
  • scale: 크기 (default: 1)
  • ease: 애니메이션의 속도를 제어 (default: 'power1.out')
  • backgroundColor, width, height, color 등 css 속성 (camelCase로 작성)
  • paused: 애니메이션을 즉시 실행할지 말지 (default: false)

gsap.from(), fromTo(), set()

gsap.from()은 to()와 정반대로, 시작값을 정하고 원래대로 되돌아오는 애니메이션을 실행

gsap.from()은 from()속성이 시작값으로 세팅, to()속성이 종료값으로 세팅되어 애니메이션을 실행

set()은 애니메이션 효과 없이 즉시 실행

return : Tween

gsap.timeline()

애니메이션을 순차적으로 실행하고싶을 때 모든 애니메이션에 delay속성을 사용하는 방법은 비효율적이다.
만약 첫번째 애니메이션을 더 길게 만들고 싶으면 그 이후의 애니메이션의 delay값을 모두 조정해야하기 때문이다.
gsap.timeline()을 사용하면 애니메이션의 순차적 실행을 간단하고 효율적으로 관리할 수 있다.
