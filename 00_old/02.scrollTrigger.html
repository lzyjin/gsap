ScrollTrigger 사용

scrollTrigger: 스크롤을 해서 해당 요소가 보일 때 애니메이션이 나타나도록 하기

scrollTrigger속성값: 애니메이션 실행 기준이 되는 요소

		gsap.to('.c', {
			scrollTrigger: '.c',
			x: 400,
			rotation: 360,
			duration: 3
		});
		---> .c 요소가 보이기 시작할 때 .c의 애니메이션이 실행된다. 
	
		gsap.to('.b', {
			scrollTrigger: '.c',
			x: 400,
			rotation: 360,
			duration: 3
		});
		---> .c 요소가 보이기 시작할 때 .b의 애니메이션이 실행된다.

toggleActions: 4개의 토글위치에서 순서대로 제어되는 방식을 결정한다

onEnter, onLeave, onEnterBack, onLeaveBack 순서

들어갈 때, 나갈 때, 뒤로 다시 들어갈 때, 처음부터 끝까지 스크롤할 때 순서

default: play none none none

markers: 트리거의 시작과 끝을 표시, 스크롤러의 시작과 끝을 표시

값: Object | Boolean

true로 할 경우 기본적으로 startColor: "green", endColor: "red", fontSize: "16px", fontWeigth: "normal", indent: 0

markers: {startColor: "white", endColor: "white", fontSize: "18px", fontWeight: "bold", indent: 20} 처럼 정의해서 커스텀할 수 있다

start: ScrollTrigger의 시작위치를 결정한다

값은 문자열, 숫자, 함수

start: "top center": 트리거의 상단이 스크롤러의 중앙에 닿을 때 시작

start: "bottom 80%": 트리거의 하단이 뷰포트 상단에서 80% 아래로 떨어질 때 시작

문자열 값: top, bottom, center, (left, right), 100% (백분율), 100px(픽셀값)
