ScrollMagic 기본 문법

1. ScrollMagic 애니메이션을 실행하기 위한 Controller 생성.
var controller = new ScrollMagic.Controller();

2. 스크롤될 때 필요한 애니메이션 오브젝트 생성.
GSAP(TweenMax) 라이브러리를 사용하여 애니메이션 코드 작성.
var tween = TweenMax.to('#animate', 0.5, { x: 600, y: 500, backgroundColor: '#6431ff', scale: 2.5, rotation: 360, });

3. ScrollMagic 상세 옵션 설정을 위한 Scene 오브젝트 생성.
var scene = new ScrollMagic.Scene();

4. Scene 오브젝트에 어떠한 객체에서 동작해야 할 상세 스크롤(애니메이션 등)을 위한 옵션 설정을 한다.
n1)어떠한 객체에서 Scene이 발생할지 트리거를 설정해야 한다. = 어떠한 객체에 스크롤이 닿으면 애니메이션이 발생할지 트리거를 설정함.
트리거 설정(Scene 메서드 옵션): triggerElement: 'class 또는 id'
2) GSAP(TweenMax) 애니메이션 오브젝트를 Scene 오브젝트에 추가한다.
애니메이션 실행 설정 메서드: setTween()
3) 마지막으로 ScrollMagic Controller에 Scene 오브젝트를 추가한다.
Controller 실행 메서드: addTo()
4) 디버깅을 위해서 인디케이터를 추가한다. 디버깅 후에는 삭제한다.
디버깅 실행 설정 메서드: addIndicators()



duration (scene 옵션)

트리거의 애니메이션 시작지점부터 종료지점까지 재생시간을 나타내는 scene 메서드의 옵션
단위는 숫자,
예) duration: 100 -> y축으로 100px 스크롤되는 동안 애니메이션 실행 예) duration: 100% -> 화면 높이에 따라 유동적으로 재생시간 변화
